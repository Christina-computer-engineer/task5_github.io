# Task 5: Deploy a Static Website Using GitHub Pages

## Objective

Host a static website online for free using GitHub Pages.

## Project Description

This project demonstrates the deployment of a static website using GitHub Pages. The website is hosted directly from a GitHub repository and can be accessed through a public URL.

## Tools Used

* GitHub

* GitHub Pages

* HTML5

* CSS3

## Steps Performed

1. Created a GitHub repository.

2. Uploaded website files (HTML, CSS, and images).

3. Enabled GitHub Pages in the repository settings.

4. Selected the main branch and root folder as the deployment source.

5. Deployed the website and verified the live URL.
